# Student Performance Predictor 📊

This project predicts student marks based on study hours and attendance using Machine Learning.

## 🔧 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🚀 Features
- Predicts marks using Linear Regression
- Shows data visualization
- Takes user input
- Calculates model accuracy
- Saves trained model using joblib

## 📈 Output Example
Predicted Marks: 63.4  
Accuracy: 0.91

## 📌 Learning Outcome
Built a complete ML pipeline from data processing to prediction and evaluation.
